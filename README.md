# Node.js,express.js with MongoDB E-commerce Project

## Overview

This is an e-commerce API that aims to assists an online shopping platform build using Express and Mongo database.
Here it contains all the required functionalities of fullfledged E-commerce API like user registration,login,authentication,
category add and delete,product add,update and delete,Add product images,order creation with payment synchronisation etc...

## Features

- User authentication and authorization
- Product catalog with search and filtering
- Shopping cart functionality
- Order processing and payment integration with stripe
- User profile management
- Admin panel for managing products, orders, and users
- Rating and comment on products

## Installation

1. Clone the repository:

   
 $  git clone https://github.com/renukashiva/ecommerce.git

2. Installation dependencies:

 $  cd ecommerce
 $  npm install
 
3. Set up environment variables:

   Create a .env file in the root directory and add the necessary environment variables:

   PORT=8080
   DATABASE_URL=mongodb://localhost:27017/ecommerce
   SECRET_KEY=your_secret_key

4. Run the application:

   npm start

   
Usage:

   Visit http://localhost:3000 in your web browser.
   Sign up or log in to start using the application.
   Browse the product catalog, add items to your cart, and proceed to checkout.
